# Web Application Vulnerability Scanner (WVS)

This tool crawls a target website and scans for basic vulnerabilities like SQL Injection, XSS, and Local File Inclusion.


## Features

- Automatic crawling

- Vulnerability testing

- Report generation


## Installation

```bash

git clone https://github.com/<your_username>/wvs-scanner.git

cd wvs-scanner

pip install -r requirements.txt

cd WVS-Src

python W-vuln.py
